# Work-Life Balance Score Predictor

**Mission:** As a software engineer focused on older adult wellbeing, this project builds a machine learning model to predict an individual's Work-Life Balance Score from daily lifestyle indicators such as sleep, stress, meditation, social connections, and physical activity.

**Dataset:** Wellbeing and Lifestyle Data — Kaggle (15,972 rows | 24 columns) | https://www.kaggle.com/datasets/ydalat/lifestyle-and-wellbeing-data

**Models:** Linear Regression (SGD), Decision Tree, and Random Forest — the best-performing model is saved and used to power a prediction API.

**Goal:** To identify which lifestyle habits most influence wellbeing scores, enabling data-driven digital solutions that help older adults live more fulfilling, connected, and healthy lives.
